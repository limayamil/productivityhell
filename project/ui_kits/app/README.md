# Productivity Hell — App UI Kit

## Overview
Mobile-first interactive prototype covering all core screens of the Productivity Hell app.

## Screens
1. **Dashboard** — Active round with tasks, timer, score, perks
2. **Task Creation Modal** — Quick-add with category/priority/time selectors
3. **Perk Selection** — Post-round reward screen (choose 1 of 3)
4. **Round Summary** — End-of-round breakdown with rank
5. **Day View** — Full day as a series of completed/active rounds

## Files
- `index.html` — Main prototype entry (interactive, mobile viewport)
- `Dashboard.jsx` — Active round screen
- `TaskCard.jsx` — Individual task component
- `PerkCard.jsx` — Perk card component
- `TaskModal.jsx` — Task creation modal
- `PerkSelection.jsx` — Perk reward selection screen
- `RoundSummary.jsx` — Round end summary screen
- `DayView.jsx` — Full day overview screen

## Usage
Open `index.html` in a browser. Navigate between screens using the bottom nav and in-app buttons.
Design width: 390px (iPhone 14). Scales to desktop via centering.
